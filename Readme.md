# Ambulance Positioning System using Machine Learning

## Project Overview
This project predicts optimal ambulance deployment locations using machine learning techniques. The system analyzes accident data and clustering patterns to reduce emergency response times.

## Features
- Accident data analysis
- Ambulance location prediction
- User registration and login
- Prediction results visualization
- Profile management

## Technologies Used
- Python
- Machine Learning
- Django
- HTML
- CSS
- SQLite
